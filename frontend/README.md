# Tyre Category Management System

## 📌 Project Overview

The **Tyre Category Management System** is a web application that helps manage different tyre categories efficiently. It allows users to view, filter, and manage tyres based on vehicle type and tyre type.

This project is built to make **tyre inventory management simple, searchable, and organized**.

---

## 🚀 Features

* 🔍 **Easy Search and Filter System**
* 🚗 **Classification by Vehicle Type**

  * Car
  * Bike
  * Truck
  * Bus
* 🛞 **Tyre Type Categories**

  * Tubeless
  * Radial
  * Tube
* 📊 **View Tyre Inventory**
* 🏷 **Brand-wise Tyre Listing**
* 📱 **Responsive User Interface**
